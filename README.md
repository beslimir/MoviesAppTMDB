# MoviesAppTMDB

This app uses the TMDB in order to provide a list of movies, artists and TV shows.
The purpose of this app is to demonstrate the use of UseCase-s in an MVVM architecture.

This app uses:

- MVVM architecture
- Domain layer (Use cases)
- Dagger 2
- Glide
- Retrofit
- LiveData
- Coroutines
- Room
- RecyclerView
